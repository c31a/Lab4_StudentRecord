# Lab4_StudentRecord
C# console application implementing a Student class with private fields, properties, GPA validation, and exception handling.

## Description
A simple C# program that demonstrates object-oriented programming concepts.

## Features
- Student class with private fields (name, id, gpa)
- Properties to access private fields
- GPA validation (must be between 0 and 4)
- Exception handling using try-catch
- DisplayRecord() method to print student details

## How to Run
1. Open the project in Visual Studio
2. Press F5 to run
